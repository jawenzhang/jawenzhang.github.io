nodejs-https几个小问题

  1. express创建https server 

    createServer.  .. option
var app = express();
var fs = require('fs');
var options = {
    key: fs.readFileSync('./mykeys/codepower.key'),
    cert: fs.readFileSync('./mykeys/codepower.crt')
};
var server = https.createServer(options, app);`

  2. https server默认端口443,直接访问https://www.codepower.cc 会连接443端口

  3. 腾讯云中申请的ssl免费证书，可以使用nginx目录中的key ,crt 两个文件，即可。
